## ğŸ“± How to Use

### Mark a Badge as "In Progress"

**Method 1: Swipe from Left**
1. Swipe **right** on any badge (swipe from left edge)
2. Tap the blue **"In Progress"** button
3. Badge is marked as started with today's date

**Method 2: From Detail View**
1. Open any badge that's not started
2. Tap the **"Mark In Progress"** button at the top
3. Badge status updates immediately

**Method 3: Check a Requirement**
1. Open badge detail view
2. Check off any requirement
3. Badge automatically marks as "In Progress"

### Remove "In Progress" Status

**Swipe from Left (Same gesture)**
1. Swipe **right** on an in-progress badge
2. Tap the gray **"Not Started"** button
3. Badge returns to not started (keeps any checked requirements)
